What happens next

Your home test kits should arrive within 2 days.

If there are any delays, call 119 and the contact centre can help.

If you’re in a bubble, make sure the other household gets their rapid tests.

What's in the pack

The pack contains:

  • 7 rapid lateral flow tests
  • a guide to twice weekly testing
  • instructions on how to take the test
  • instructions on how to report each result to the NHS

When the test pack arrives

Your household (bubble) should:

  • take a test twice a week (every 3 or 4 days apart)
  • report every result to the NHS on the same day you take the test
  • try to test as one household - on the same day and around the same time
