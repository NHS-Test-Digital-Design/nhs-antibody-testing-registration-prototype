Test with assistive technology

The service must be tested with commonly used assistive technologies and any issues fixed. These include screen magnifiers, screen readers and speech recognition tools.

  • Met

    Testing has taken place with all of the commonly used assistive technologies and any issues fixed.

  • In progress

    Testing has taken place with some of the commonly used assistive technologies but not all of them. Or testing has taken place but the issues have not been fixed.

  • Not met

    No testing has taken place or unknown technologies have been used.
