A test registration code is unique to each person. It stores their personal details so that you do not have to re-enter them each time you register a test for them.

What it looks like

You or someone in your organisation may have created a test registration code for this person when you registered a test for them in the past.

It is 8 characters long, for example ABC12345. Find it on emails and texts NHS Test and Trace has sent you.
