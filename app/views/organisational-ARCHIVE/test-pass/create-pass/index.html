Create test registration codes for your organisation

Use this service to create test registration codes for people at your organisation. When you register a test for someone, entering their code will save you time by automatically adding their personal details to your registration.

How test registration codes work

When you create a test registration code for someone, we securely store the personal details you give us so they can be used in future. These details can only be accessed using the code.

Make sure you:

  • keep a record of each code - we'll also send it to you by email or text
  • get consent from each person to create a test registration code for them and to have their code sent to the contact details you enter

To find out more information on test registration codes, view the test registration codes guidance.

What you need

  • your 8 digit unique organisation number (UON)
  • the personal details for each person you’re creating a test registration code for
  • if you would like to create codes for multiple people, using the test registration code spreadsheet will save you time

How we use personal data

To find out how we use people’s data when you create a test registration code for them, read the coronavirus (COVID-19) privacy note.
