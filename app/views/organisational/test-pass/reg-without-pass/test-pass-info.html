Test registration codes will save you time when registering tests or reporting results through this service.

Overview

A test registration code is a unique 8 digit number that is linked to an individual at your organisation. You can create test registration codes for both residents and staff.

When you create and enter a test registration code for someone, information such as their contact details, address and NHS number will be saved automatically. You will no longer have to enter these details every time you register a test or report a result on their behalf.

You will still need to enter the person’s first name, last name, date of birth and test details each time you register a test for them.

How to create and use test registration codes

If you are registering tests using the spreadsheet:

  1. Fill in the spreadsheet with each person’s details.
  2. Upload the spreadsheet.
  3. Once you have checked the test details, you will be asked if you would like to create test registration codes for each person on your spreadsheet.
  4. Select the people you would like to create test registration codes for.
  5. You will receive an email and text for each test registration code you create – these will go to the contact details you provided, but you can also send them to an additional email address if you wish.
  6. The next time you register a test for them, you can enter their test registration code in the spreadsheet to save time.

If you are registering tests one at a time:

  1. Enter the person’s details.
  2. Once have you have completed their information, you will be asked if you would like to create a test registration code for that person.
  3. You will receive an email and text for each test registration code you create – these will go to the contact details you provided, but you can also send them to an additional email address if you wish.
  4. The next time you register a test for them, you can enter their test registration code to save time.

Keep a record of each test registration code you create so that you can enter it the next time you use this service.
