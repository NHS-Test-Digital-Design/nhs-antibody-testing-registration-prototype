What is a test registration code?
A test registration code is a unique 8 digit number that is linked to an individual at your organisation. When you create a test registration code for someone, information such as their contact details, address and NHS number will be saved automatically. You will no longer have to re-enter these details every time you register a test or report a result on their behalf.
