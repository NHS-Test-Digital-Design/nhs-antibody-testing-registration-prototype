Create test registration codes for your organisation

Register test kits faster by creating test registration codes for people at your organisation. When you use the codes, it’ll automatically add their personal details, saving time.

How test registration codes work

  • when we create a unique test registration code for a person, we securely store the personal details you give us. They can only be accessed with this code
  • you need to make a note of the code. We’ll also send it to you by either email or text
  • the next time you register a test kit for the person, enter their test registration code
  • this automatically fills in their personal details, saving you time
  • if you forget their code you can set up a new one

Get consent

Make sure people getting test registration codes have freely given you consent to:

  • create a code for them
  • get their test registration code sent to the contact details you enter

What you'll need

  • The personal details for each person you're creating a test registration code for
  • The test registration code spreadsheet, this will save you time if you need to create several test registration codes at the same time

How we use personal data

To find out how we use people's data when you register them for a test registration code, read the coronavirus privacy note.
