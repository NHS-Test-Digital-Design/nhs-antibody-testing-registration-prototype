Receiving test registration codes

Organisations should keep a record of each test registration code they create so they can be used for future registrations. You will be able to view and download a list of all the test registration codes created at the end of this registration process

Each person’s test registration code will be sent to the contact details you provided earlier (the email address and mobile number where their test results are being sent).

If you would like to send all the test registration codes to an additional person (for example, your organisation’s testing manager), enter their contact details here. You can also choose to send each person’s test registration code to separate additional contacts instead.
