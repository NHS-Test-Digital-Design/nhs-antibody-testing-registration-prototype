This accessibility statement applies to a range of coronavirus test ordering and registration services for organisations. Please note they are not available to the general public.

We want as many people as possible to be able to use these services. For example, this means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • complete the service using just a keyboard
  • complete the service using speech recognition software
  • complete the service using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made all text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible are these services?

We are aware that some parts of these services for organisations are not fully accessible. We are working as quickly as we can to improve this. Examples include:

  • The screen asking for the patient’s occupation is not accessible to keyboard only and screen reader users
  • The reCAPTCHA security check is not fully accessible to those using screen readers
  • Some fields and links are not accessible to screen reader users because of insufficient labelling
  • The error message boxes at the top of the screen are difficult to use for keyboard only users

We are committed to making sure online services are delivered in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Complaints

If you have feedback or a complaint about the accessibility of this service, email us scas.Covid19TestingComplaints@nhs.net.

Compliance status

Our website-based services are partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard and we are working to make the necessary improvements.
