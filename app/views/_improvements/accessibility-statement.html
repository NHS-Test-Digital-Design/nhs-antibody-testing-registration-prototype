This accessibility statement applies to these coronavirus testing services:

  • Organisation user managemen
  • Super user portal
  • Trust user portal
  • Patient home linking journey
  • Bulk registration

We want as many people as possible to be able to use these services. For example, this means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • complete the service using just a keyboard
  • complete the service using speech recognition software
  • complete the service using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made all text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible is this service?

We are aware that some parts of the service are not fully accessible. We are working as quickly as we can to improve this.

Complaints

If you have feedback or a complaint about the accessibility of this service, email us at scas.Covid19TestingComplaints@nhs.net.

Contacting us by phone

If you have questions about a test you've booked or are having trouble booking a test, call the customer contact centre from 7am to 11pm.

  • England and Wales: call 119 (free from mobiles and landlines)
  • Scotland and Northern Ireland: call 0300 303 2713 (charged at your standard network rate)

Technical information about this service’s accessibility

We are committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Compliance status

This website is partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard and we are working to evaluate where we are non-compliant.

Preparation of this accessibility statement

This statement was prepared on 3rd December 2020. It was last reviewed on 3rd December 2020.

We are aware that some parts of the websites are not fully accessible. We are working as quickly as we can to improve this and will fully update this statement in due course.
